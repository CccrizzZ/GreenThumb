> Canabis cultivation monitor/planner build with electron-vue

#### Build Setup

``` bash
# cd to folder

# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
